# ☠ Pirate Battleship

Морской бой в пиратском стиле, построенный на **React + TypeScript + Three.js**.

## Стек

| Технология | Назначение |
|---|---|
| React 18 + TypeScript | UI, игровая логика, состояние |
| Three.js + @react-three/fiber | 3D-сцена с кораблями и водой |
| @react-three/drei | OrbitControls, хелперы |
| GLSL шейдеры | Анимированная WebGL-вода |
| Vite | Сборщик и dev-сервер |

## Запуск

```bash
npm install
npm run dev
```

Открыть в браузере: [http://localhost:5173](http://localhost:5173)

## Архитектура

```
src/
  game/
    types.ts          — типы: Cell, Board, Ship, GameState, LogEntry
    logic.ts          — чистая логика игры (расстановка, атаки, AI)
  components/
    Board.tsx         — 2D-сетка 10×10, анимации попаданий/промахов
    Board.module.css
    Hud.tsx           — статус, лог, счёт, иконки кораблей
    Hud.module.css
  three/
    SeaScene.tsx      — Three.js сцена: вода + процедурные корабли
  App.tsx             — главный компонент, связь логики и UI
  App.module.css
  styles/global.css
  main.tsx
```

## Возможности

- 🌊 Анимированная WebGL-вода на GLSL-шейдерах (вершинный + фрагментный)
- ⚓ Процедурные 3D-корабли от 1 до 4 клеток в пиратском стиле (мачты, паруса, пушки, флаг)
- 💥 3D-взрывы при попадании + анимация потопления (наклон, опускание)
- 🎮 Полноценная логика морского боя 10×10 с умным AI (целится по соседним клеткам)
- ⚡ React + TypeScript, CSS Modules, современный пиратский дизайн
